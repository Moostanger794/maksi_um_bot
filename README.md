# Telegram-бот MaksiUm (aiogram + SQLite)

## Возможности

- Запись родителя на занятие через inline-календарь (на 30 дней вперед).
- Выбор свободного времени и бронирование только 1 слота на пользователя.
- Автоматическая блокировка занятого слота.
- Сбор данных: имя родителя, имя ребенка, телефон.
- Отправка записи директору и в канал с общим расписанием.
- Отмена своей записи с освобождением слота.
- Проверка подписки на канал перед записью (опционально).
- Кнопки: "О центре", "Направления и цены", "Фотогалерея".
- Админ-панель (по ID): добавление/удаление слотов, закрытие/открытие дня, просмотр записей по дате, отмена записи клиента.
- Автонапоминания за 24 часа через APScheduler.
- Восстановление задач напоминаний из БД после перезапуска.

## Настройка

1. Установите Python 3.11+.
2. Установите зависимости:

   ```bash
   pip install -r requirements.txt
   ```

3. Создайте переменные окружения (можно через `.env`, через PowerShell или системные переменные):

- `BOT_TOKEN` - токен бота.
- `ADMIN_ID` - Telegram ID директора/админа.
- `CHANNEL_ID` - ID канала (например, `-1001234567890`). Если `0`, бот работает без канала.
- `CHANNEL_LINK` - ссылка на канал для кнопки подписки.
- `GALLERY_URL` - ссылка на фотоальбом.
- `DB_PATH` - путь к SQLite-файлу (по умолчанию `maksium.db`).

Пример:

```powershell
$env:BOT_TOKEN="123456:ABC"
$env:ADMIN_ID="123456789"
$env:CHANNEL_ID="0"
$env:CHANNEL_LINK="https://t.me/your_channel"
$env:GALLERY_URL="https://www.instagram.com/maksium_tambov/"
$env:DB_PATH="maksium.db"
```

4. Запуск:

```bash
python bot.py
```

## Важно перед использованием

- Если `CHANNEL_ID=0`, бот работает полностью автономно, без участия в канале.
- Для корректного расписания админу нужно добавить слоты через админ-панель.
- Напоминание ставится только если до занятия больше 24 часов.
